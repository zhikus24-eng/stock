# KCEX 美股市場 Demo

這是一個純前端的美股市場資訊與交易介面 Demo，不需要安裝套件或建置工具。

## 專案結構

```text
.
├── index.html          # 頁面結構與資源入口
├── css/
│   └── styles.css      # 全站樣式與響應式版面
└── js/
    ├── glossary.js     # 專有名詞提示與說明
    ├── stock-data.js   # 股票資料與共用狀態
    ├── market.js       # 清單、搜尋、排行與市場模組
    ├── trading.js      # 詳情抽屜、財務資訊與交易面板
    └── share.js        # 分享視窗、提示訊息與導覽互動
```

## 使用方式

直接用瀏覽器開啟 `index.html` 即可預覽。

若瀏覽器對本機檔案有限制，也可以在專案目錄啟動任一靜態網站伺服器，例如使用 VS Code 的 Live Server。

## 注意事項

- 所有行情與公司資料皆為前端示範資料，不代表即時市場資訊。
- 交易按鈕與部分互動為 Demo 或外部連結，不會在此專案中送出真實訂單。
- 介面圖標使用 Font Awesome CDN，離線開啟時需有網路連線才能顯示圖標。
- 各 JavaScript 檔案依 `index.html` 內的順序載入，調整檔案順序時需留意模組之間的資料相依性。
